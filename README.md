FACE RECOGNITION (PYTHON)

A simple face recognition program built using Python.
This project detects human faces in real time and draws a red square around each detected face.

FEATURES

Built in Python

Detects faces in images or live camera feed

Draws a red bounding box around detected faces

Runs locally

REQUIREMENTS

Before running this project, make sure you have:

Python 3 installed

A webcam (if using live detection)

INSTALLATION AND SETUP

INSTALL PYTHON

If Python is not installed, download and install it from:

https://www.python.org

Verify installation by running:

python --version

INSTALL REQUIRED LIBRARIES

Install the required dependencies using pip:

pip install opencv-python

RUN THE PROJECT

Navigate to the project directory and run:

python main.py

(Replace main.py with the correct file name if different.)

OUTPUT

The program will detect faces and display a red square around each recognized face in real time.

TECH STACK

Python

OpenCV

PROJECT PURPOSE

This project was created to:

Learn basic computer vision concepts

Practice face detection in Python

Demonstrate image processing skills
